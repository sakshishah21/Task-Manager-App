
**Frontend :-**
React.js
Axios for API calls
Tailwind CSS for styling
Material UI (MUI) Components
React Hooks & Functional Components

**Backend :-**
Node.js + Express.js
MongoDB + Mongoose
JWT Authentication
Multer for Image Upload
Bcrypt Password Encryption

**Image Storage :-**
Stored in /uploads folder (or database, depending on your setup)

**Installation & Setup :-**
Clone the Repository using below command
git clone https://github.com/sakshishah21/Task-Manager-App.git

**:- run backend project :-**
cd backend (go to back end project path)
npm install (required project install)
setup .env file (database name, port setup)
npm run dev (using this command run back-end)

**:- frontend :-**
cd frontend (go to front end project)
npm install (install required packages)
npm run start (run front end project using this command)
